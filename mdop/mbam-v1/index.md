# Guide d’administration et d’administration de Microsoft BitLocker 1

Microsoft BitLocker Administration and Monitoring (MBAM) fournit une interface d’administration simplifiée que vous pouvez utiliser pour gérer le chiffrement de lecteur BitLocker. Avec MBAM, vous pouvez sélectionner des options de stratégie de chiffrement BitLocker appropriées à votre entreprise, puis les utiliser pour contrôler la conformité du client avec ces stratégies. Vous pouvez également signaler l’état de cryptage d’un ordinateur individuel et de l’ensemble de l’entreprise. De plus, vous pouvez accéder aux informations de clé de récupération lorsque les utilisateurs oublient leur code confidentiel ou leur mot de passe, ou lorsque leur enregistrement de BIOS ou de démarrage est modifié.
