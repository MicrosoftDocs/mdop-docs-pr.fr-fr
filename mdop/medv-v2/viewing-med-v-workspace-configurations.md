# Affichage des configurations d'espace de travail MED-V


Lorsque vous rencontrez des problèmes et rencontrez des problèmes avec la gestion des problèmes de MED-v, vous pouvez utiliser le kit de tâches d’administration MED-V pour afficher les applications publiées et les adresses Web redirigées dans l’espace de travail MED-V. Vous pouvez également utiliser le kit de tâches d’administration MED-V pour ouvrir l’espace de travail MED-V en mode plein écran.

Pour plus d’informations sur l’ouverture du kit de tâches d’administration MED-V, voir [résolution des problèmes de med-v à l’aide du kit de tâches d’administration](troubleshooting-med-v-by-using-the-administration-toolkit.md).

## Affichage des applications publiées MED-V


Dans la fenêtre **Kit de tâches d’administration de MED-V** , cliquez sur Afficher les **applications publiées**.

La fenêtre **applications d’espace de travail MED-V** s’ouvre et affiche la liste des applications qui ont été publiées dans l’espace de travail MED-v. Vous pouvez utiliser ces informations pour résoudre certains problèmes, par exemple pour déterminer si une application a été publiée comme prévu.

Pour plus d’informations sur la publication d’applications dans l’espace de travail MED-V, voir [Comment publier et annuler la publication d’une application dans l’espace de travail MED-v](how-to-publish-and-unpublish-an-application-on-the-med-v-workspace.md).

## Afficher les adresses Web redirigées MED-V


Dans la fenêtre **Kit de tâches d’administration de MED-V** , cliquez sur Afficher les **adresses Web redirigées**.

Les **adresses Web redirigées vers la fenêtre d’espace de travail MED-v** s’ouvre et affiche la liste des adresses Web redirigées spécifiées pour l’espace de travail MED-v. Vous pouvez utiliser ces informations pour résoudre certains problèmes, par exemple pour déterminer si une adresse Web a été correctement spécifiée pour la redirection.

Pour plus d’informations sur la gestion de la redirection d’adresses Web dans l’espace de travail MED-V, voir [comment ajouter ou supprimer des informations de redirection d’URL dans un espace de travail MED-v déployé](how-to-add-or-remove-url-redirection-information-in-a-deployed-med-v-workspace.md).

## <a href="" id="bkmk-fullscreen"></a>Ouverture de l’ordinateur virtuel de l’espace de travail MED-V


Dans la fenêtre du **Kit de tâches d’administration de med-v** , cliquez sur **afficher l’intégralité de l’espace de travail MED-v**.

MED-V se ferme s’il était en cours d’exécution et l’ordinateur virtuel de l’espace de travail MED-V s’ouvre en mode plein écran. Vous pouvez utiliser cette fenêtre en mode plein écran pour accéder facilement à tous les composants de l’ordinateur virtuel qui peuvent être utiles lors de la résolution des problèmes, tels que les fichiers de configuration et de disque dur.
