## Modèles d’emplacement des paramètres d’Office 2007


Le logiciel d’installation de l’agent UE-V installe l’agent et inscrit un groupe par défaut de modèles d’emplacements des paramètres pour les applications Microsoft courantes. Microsoft Office 2007 fait désormais partie de ces applications. Il existe deux modèles Office 2007: MicrosoftOffice2007.xml et MicrosoftCommunicator2007.xml. Ces paramètres de capture de modèles d’emplacement des paramètres dans Microsoft Office 2007 pour les applications suivantes:

-   Microsoft Access 2007

-   Microsoft Communicator 2007

-   Microsoft Excel 2007

-   Microsoft InfoPath 2007

-   Microsoft OneNote 2007

-   Microsoft Outlook 2007

-   Microsoft PowerPoint 2007

-   Microsoft Project 2007

-   Microsoft Publisher 2007

-   Microsoft SharePoint Designer 2007

-   Microsoft Visio 2007

-   Microsoft Word 2007

### Mises à jour de modèles d’emplacement des paramètres d’Office 2010

Une mise à jour des modèles d’emplacement des paramètres a également été effectuée. Ces modifications sont les suivantes:

-   Ajout de la prise en charge de Microsoft SharePoint Designer 2010 en ajoutant un nouveau modèle aux modèles Office 2010 (MicrosoftOffice2010Win32.xml et MicrosoftOffice2010Win64.xml)

-   Correction de bogues mineurs, notamment personnaliser la barre d’État – Word, Excel et PowerPoint

## La tâche planifiée pour les mises à jour de catalogue est désormais aléatoire


La tâche de mise à jour automatique de modèle vérifie dans le catalogue de modèles de paramètres les modèles nouveaux, mis à jour ou supprimés. Cette tâche s’exécute uniquement si la SettingsTemplateCatalog est configurée. La tâche de mise à jour automatique du modèle exécute le fichier ApplySettingsCatalog.exe qui se trouve dans le répertoire d’installation de l’agent UE-V et où UE-V SP1 a été modifié de manière à aléatoirer la mise à jour sur une période d’une heure.

## Prise en charge de Citrix EdgeSight


Un conflit a été détecté avec UE-V exécuté sur un serveur doté de Citrix EdgeSight. UE-V 1,0 SP1 résout ce problème.

## Indexation des favoris d’Internet Explorer


Lorsque UE-V itinérance des favoris Internet Explorer d’un ordinateur vers un autre, l’indexation des adresses favorites dans la barre d’adresses de l’ordinateur synchronisé est désormais mise à jour. Lorsqu’un utilisateur tape dans la barre d’adresses, les favoris itinérants apparaissent désormais comme disponible sur les ordinateurs synchronisés.

## Nouveaux paramètres de ligne de commande setup.exe pour UE-V agent et UE-V Generator


Avec la publication d’UE-V 1,0 SP1, le setup.exe pour l’agent UE-v et le générateur UE-V a été mis à jour de manière à permettre les paramètres de ligne de commande supplémentaires suivants:

1.  `CEIPENABLED` – Permet à l’utilisateur d’accepter l’option d’inclusion du programme d’amélioration du produit de Microsoft.

2.  `INSTALLFOLDER` – Permet de définir un autre dossier d’installation pour l’agent ou le générateur.

3.  `MUENABLED` – Permet à l’installation d’accepter l’option d’inclusion du programme Microsoft Update.
