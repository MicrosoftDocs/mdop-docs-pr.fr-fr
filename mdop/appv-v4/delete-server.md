# DELETE SERVER


Supprime un serveur de publication.

**Remarques**  Cette commande n’entraîne pas la suppression des applications ou packages publiés sur le client par le serveur. Pour chaque application, utilisez la commande SFTMIME **Clear app** suivie de la commande **supprimer le package** pour supprimer complètement ces applications et packages du client.

 

`SFTMIME DELETE SERVER:server-name [/LOG log-pathname | /CONSOLE | /GUI]`
