# Procédure pour exporter un rapport


Le processus d’exportation d’un rapport à l’aide de la console de virtualisation des applications est le même quel que soit le type de votre rapport.

**Pour exporter un rapport**

1.  Cliquez avec le bouton droit sur le rapport affiché dans le volet **résultats** et sélectionnez **Exporter** dans le menu déroulant.

2.  Sélectionnez le type de rapport dans la liste des types.

3.  Dans la boîte de dialogue **Enregistrer sous** , accédez à l’emplacement où vous souhaitez enregistrer la sortie.

4.  Entrez un nom pour le rapport dans le champ **nom de fichier** , ou sélectionnez un fichier dans la liste déroulante des fichiers.

5.  Sélectionnez un type de sortie dans la liste déroulante **type de fichier** .

6.  Cliquez sur le bouton **Enregistrer** pour enregistrer le fichier.
