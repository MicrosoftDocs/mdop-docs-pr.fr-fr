# À propos de Microsoft Application Virtualization4.6 SP3


Microsoft Application Virtualization (App-V) 4.6 SP3 fournit les améliorations et nouvelles fonctionnalités suivantes décrites dans cette rubrique.

## Prise en charge de Windows Server 2012 R2 et Windows 8,1


App-V 4,6 SP3 inclut la prise en charge de Windows Server 2012 R2 et Windows 8,1

## Obtention de technologies MDOP


App-V est un composant du pack d’optimisation du bureau Microsoft (MDOP). MDOP fait partie de Microsoft Software Assurance. Pour plus d’informations sur le programme d’assurance logiciel Microsoft et sur l’acquisition de MDOP, voir [Comment obtenir MDOP](https://go.microsoft.com/fwlink/?LinkId=322049) ( https://go.microsoft.com/fwlink/?LinkId=322049) .
